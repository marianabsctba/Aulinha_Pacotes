Explicação simples

Pacote é como uma carta digital. Cada camada adiciona informações importantes para que ele chegue ao destino correto. Ethernet resolve o “para quem eu entrego aqui dentro”, IP resolve “para qual rede eu vou”, TCP resolve “qual conversa é essa” e o payload carrega a mensagem.

clique nas camadas para mudar a explicação
💡 Dica: cada dispositivo olha só o que precisa. Switch olha MAC. Roteador olha IP. Firewall pode olhar bem mais... porque ele é curioso com crachá.
